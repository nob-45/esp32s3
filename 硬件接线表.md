# SmartElderCare 硬件接线表

> 目标板：**ESP32-S3-DevKitC-1 v1.1**（乐鑫官方开发板，正面 USB-C 靠上，两侧各一排排针）
> 更新时间：2026-07-14
> 代码已编译通过（Flash 占用 4%），硬件到货后按下表接线即可，无需改代码。

## 一、LCD 屏幕 —— 正点原子 ATK-MD0240 V1.2（240×320 SPI TFT，ST7789）

| 屏丝印 | ESP32-S3 引脚 | 说明 |
|--------|--------------|------|
| **3V3** | 3.3V | 屏幕电源 |
| **GND** | GND | 地 |
| **SCK** | GPIO12 | SPI 时钟 |
| **SDA** | GPIO11 | SPI 数据 (MOSI) |
| **CS**  | GPIO10 | 片选 |
| **WR**  | GPIO13 | 数据/命令切换 ⚠️（正点原子把 DC 印成 WR，实际就是 DC） |
| **RST** | GPIO14 | 复位 |
| **PWR** | GPIO2  | 背光（PWM 调光；也可直接跳到 3.3V 常亮） |

## 二、毫米波雷达 —— HLK-LD2410B-P（24 GHz 人体存在检测）

**当前使用玩法A：只读数字电平（有人=高电平 / 无人=低电平），简单可靠。**

| 雷达引脚 | ESP32-S3 引脚 | 说明 |
|---------|--------------|------|
| **VCC** | **5V**（板子丝印 5V 引脚） | ⚠️ **必须 5V**，接 3.3V 会异常 |
| **GND** | GND | 地 |
| **OUT** | GPIO4 | 数字输出：有人=HIGH，无人=LOW |
| **TX**  | 悬空 | 玩法A 不用；后续升级 UART 时接 GPIO18 |
| **RX**  | 悬空 | 玩法A 不用；后续升级 UART 时接 GPIO17 |

**灵敏度 / 检测距离 / 无人延时**：用手机 App「HLKRadarTool」通过蓝牙一次性配置好即可，无需代码调整。

## 三、浴室积水传感器（D0 数字阈值 + A0 模拟量，双路输出）

| 传感器引脚 | ESP32-S3 引脚 | 说明 |
|-----------|--------------|------|
| **VCC** | 3.3V | 电源 |
| **GND** | GND | 地 |
| **A0**  | GPIO6 | 模拟输出（ADC1_CH5），读取实际水位强度 |
| **D0**  | GPIO5 | 数字阈值输出，有水=LOW（板载电位器调节触发阈值） |

## 四、GPIO 汇总表（打印贴板子旁边最方便）

| GPIO | 方向 | 接到哪里 | 用途 |
|------|------|--------|------|
| GPIO2  | OUT | LCD PWR | 背光 PWM |
| GPIO4  | IN  | 雷达 OUT | 人体存在电平 |
| GPIO5  | IN  | 水传感器 D0 | 水位数字阈值 |
| GPIO6  | ADC | 水传感器 A0 | 水位模拟量 |
| GPIO10 | OUT | LCD CS  | SPI 片选 |
| GPIO11 | OUT | LCD SDA | SPI MOSI |
| GPIO12 | OUT | LCD SCK | SPI 时钟 |
| GPIO13 | OUT | LCD WR  | 数据/命令（DC） |
| GPIO14 | OUT | LCD RST | LCD 复位 |
| **3.3V** | — | LCD VCC / 水传感器 VCC | 3.3V 供电 |
| **5V**   | — | 雷达 VCC | ⚠️ 雷达必须 5V |
| **GND**  | — | 全部模块 GND | **共地！** |

## 五、⚠️ 接线注意事项（务必看！）

1. **雷达必须接 5V**：HLK-LD2410B 标称 5–12V，接 3.3V 会工作不稳或不识别人。
2. **共地**：所有传感器、屏幕的 GND 都必须与 ESP32-S3 的 GND 相连，否则：
   - A0 采样值会漂
   - 雷达 OUT 电平会乱跳导致误触发
3. **GPIO2 的特殊性**：它是 Strapping 引脚，但代码里只做 **输出（背光 PWM）**，不影响启动，放心用。
4. **水传感器不要长时间浸水**：金属电极会电解腐蚀，演示完擦干；长期部署要换成 **电容式** 或 **SUS304 不锈钢探针**。
5. **雷达安装朝向**：正对被监护区域，探测角度约 ±60°，最远约 6m；离墙 ≥30cm 避免多路径反射误判。
6. **接线顺序建议**：先断电插好所有杜邦线 → 检查一遍 3.3V / 5V / GND 有没有插错 → 再插 USB-C 上电。

## 六、后续升级：雷达玩法B（UART 读实时距离和能量）

代码里已经预留 UART 接口，届时只需在雷达上额外接两根线：

| 雷达引脚 | ESP32-S3 引脚 | 说明 |
|---------|--------------|------|
| TX | GPIO18 | 雷达 → ESP32 数据 |
| RX | GPIO17 | ESP32 → 雷达 指令 |

这样就可以在 LCD 上显示「XX 米有人 / 静止 / 运动」等更丰富的状态。

## 七、上电测试步骤

1. 按上表接好所有线（**先断电！**）
2. 用 USB-C 数据线连 ESP32-S3 到电脑
3. 在项目目录运行：
   ```
   idf.py -p COM3 flash monitor
   ```
   （COM 号根据你电脑实际情况改）
4. 观察 LCD 是否正常显示：
   - 顶部标题「智慧养老网关」
   - PRESENCE 状态（OCCUPIED / EMPTY）随人体走动切换
   - 水位数值、状态条随传感器沾水变化
5. 用手在雷达前 3 秒内挥动 → LCD 应显示 OCCUPIED
6. 用湿棉签轻擦水传感器电极 → LCD 水位百分比应上升
